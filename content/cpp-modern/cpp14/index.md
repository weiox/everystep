description: "泛型 lambda、变量模板、二进制字面量等实用增强"
---

C++14 是对 C++11 的修订和增强，带来了更多语法糖和实用特性。

## 计划中的内容

- [ ] **泛型 lambda** — auto 参数让 lambda 更灵活
- [ ] **返回类型推导** — 函数也能用 auto
- [ ] **变量模板** — 模板不再局限于类型和函数
- [ ] **[[deprecated]] 属性** — 标记过时的 API
- [ ] **二进制字面量** — 0b1010 更直观
- [ ] **数字分隔符** — 1'000'000 更易读
